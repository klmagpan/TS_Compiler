# TypeScript Compilation Demo

## Branch: compiling-the-demo-application

- Invoke the TypeScript build command
    - TSC will generate JS files
- Inspect generated JS files
    
## Usage
- Install dependencies
```bash
npm install
```

- Run app
```bash
npm run dev
```

## To build with NPX
```
npx tsc
```

## Important config options
- noEmit
- outDir
